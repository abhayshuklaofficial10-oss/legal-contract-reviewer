Let’s look at how it works under the hood. 
Everything starts at the Security Checkpoint. This node scrubs PII and blocks prompt injection attempts before the data even reaches the LLM. 
If the check passes, the workflow hands off to the Risk Analyzer agent. This agent uses an MCP server to lookup complex legal terms and compare the contract against our company’s policy benchmarks.
Next, the Clause Rewriter agent takes those risks and suggest balanced, standard language.
Finally, the result is synthesized into a detailed report for the user.
